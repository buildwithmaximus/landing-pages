Program Structure

Five parallel workstreams

The migration should be managed as a program, not a single setup task. These are the five tracks that need coordination.

1. Commercial / Vendor

Penguin selected, SOW revised, milestone payments agreed, DPA path established, support model defined.

2. Privacy / Consent / Legal

External contact migration, DPA signature, data residency, AI opt-out, and formal rules for user/integration handling.

3. Technical Setup

HubSpot access, SSO, DNS / DKIM, tracking code, Adobe Analytics, office IP exclusions, and environment readiness.

4. Content / Asset Rebuild

New assets in HubSpot, not blind Marketo copying. Campaign templates, forms, landing pages, automations, and reporting setup.

5. Enablement / Go-Live

Testing, training, user onboarding, CSM coordination, marketer readiness, dashboards, and contingency planning.
