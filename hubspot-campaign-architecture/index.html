תאימות Outlook
Outlook Compatibility — Desktop, Web & Mobile
גרסת Outlook מנוע רינדור מגבלות עיקריות פתרון
Desktop (Windows) Word/HTML Hybrid אין תמיכה ב-CSS Flexbox/Grid · תמיכת CSS מוגבלת שימוש ב-Table Layout · Inline CSS · Conditional Comments
Web (OWA) WebKit תמיכת CSS טובה יותר · עדיין דורש בדיקה בדיקה עם Litmus / Email on Acid
Mobile (iOS/Android) WebKit בדרך כלל תקין · Responsive Design עובד Media Queries לעיצוב רספונסיבי

סגמנטציה לפי ניקוד
Segmentation Thresholds — MQL / SQL Definitions
טווח ניקוד סגמנט פעולה מומלצת
0–20 Cold Lead Nurture כללי — ניוזלטר + תוכן רלוונטי
21–50 Warm Lead Nurture ממוקד — תוכן והזמנות לאירועים
51–75 MQL (Marketing Qualified) התראה לצוות השיווק · העברה לבדיקת BD
76+ SQL (Sales Qualified) העברה למכירות / BD לטיפול ישיר
חשוב: ערכי הניקוד לעיל הם המלצה ראשונית. מומלץ לכייל את המודל לאחר 3-6 חודשי פעילות על בסיס נתוני המרה בפועל.
