מה יש כאן

העמוד הזה לא מנסה להציג כל פרט טכני מהמקור, אלא לארוז את כל המערכת למבנה ניהולי ברור: מה בונים, איך זה מחובר, ומה חייב להיות מוכן כדי להגיע ל-Go-Live.

שכבת המיילים המשותפת

הבסיס שמעליו כל הקמפיינים יושבים: תבנית מותגית אחת, תאימות Outlook, יכולות עריכה, וניהול דוחות.

Base Template

תבנית מייל בסיסית

  • Header ממותג, Pre-header, Hero image, H1, body, CTA ו-footer קבוע
  • תמיכה ב-personalization tokens
  • בנויה לשימוש חוזר בכל סוגי הקמפיינים
Outlook

תאימות Outlook

  • Desktop Windows: table-based layout + inline CSS + conditional comments
  • Web / Mobile: תמיכה טובה יותר, אבל עדיין דורש בדיקות
  • Testing חובה עם Litmus / Email on Acid
Editing

יכולות עריכה

  • Drag & Drop לעבודה שוטפת של שיווק
  • HTML / coded templates רק למקרים מיוחדים
  • Smart Content, A/B testing, preview & test
Reporting

מדדי מייל מרכזיים

  • Delivery, Open, CTR, CTOR
  • Bounces, Unsubscribe, Spam Rate
  • Heat Map ו-Device Breakdown

מודל ניקוד — שלב עתידי

לא חלק מה-Go-Live, אבל כן מפת דרכים חשובה לבשלות השיווקית בהמשך.

מומלץ להתחיל לכייל את המודל אחרי 3–6 חודשי פעילות, כשיש כבר דאטה התנהגותי אמיתי ב-HubSpot.
